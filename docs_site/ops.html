Operations Guide

Running in Hosted Mode

In hosted mode, the dashboard reads from pre-generated demo CSVs under docs_site/demo_data/, built offline from two CC0 Kaggle datasets:

  • Credit: “Give Me Some Credit” (loan-level default dataset)
  • Fraud: “Credit Card Transactions Fraud Detection Dataset” (transaction-level fraud labels)

The public site shows a fixed 30-day snapshot. In local mode, a daily scheduler regenerates these KPIs from fresh scoring output under docs_global/bi/.

Directory Structure

docs_site/
├── demo_data/
│   ├── credit/
│   │   └── kpis_daily.csv
│   └── fraud/
│       ├── kpis_daily.csv
│       └── metrics_daily.csv

Running in Local Mode

In local mode, the dashboard reads from your pipeline outputs in the docs_global/ directory.

Required Directory Structure

your_repo/
├── docs_site/ (this dashboard)
└── docs_global/
    ├── bi/
    │   ├── credit/
    │   │   └── kpis_daily.csv
    │   └── fraud/
    │       ├── kpis_daily.csv
    │       └── metrics_daily.csv
    └── reports/
        ├── credit/
        │   └── YYYY-MM-DD/
        │       └── credit_daily_report.html
        └── fraud/
            └── YYYY-MM-DD/
                └── fraud_daily_report.html

Serve the site from the repo root: python -m http.server
